KarmaWorld Server Kit - Clean Distribution

This package is a clean public distribution of the Server Kit.
Personal configuration, passwords, tokens, webhooks, player history and local data have been removed.

Included:
- app.pyw -> local backend
- Iniciar_KarmaWorld.bat -> starts the manager
- Instalar_Modo_Escritorio.bat -> optional desktop mode
- config/config.json -> main configuration template
- config/panel_users.json -> first panel admin template
- static themes -> dark, ocean, emerald, violet and sunset

Not included:
- personal passwords / tokens / webhooks
- personal player history / inventory history
- personal teleports / VIP data
- personal backups / cached data

Important:
- The first panel admin is created manually in config/panel_users.json.
- If you enable panel login, create the admin first.
- Configure your own server paths before using the kit.

Note:
This ZIP has been cleaned and prepared for sharing. For stronger protection, distribute a compiled EXE build and keep the source private.
